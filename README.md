# 🧮 Modern Web Calculator

A clean, responsive, and fully functional web calculator built with vanilla **HTML5, CSS3, and JavaScript**. 

---

## 🚀 Features

* **Basic Arithmetic:** Addition, subtraction, multiplication, and division.
* **Responsive Design:** Optimized for both desktop and mobile screens.
* **Keyboard Support:** Input numbers and operators directly from your physical keyboard.

---

## 🛠️ Tech Stack

* **Structure:** HTML5
* **Styling:** CSS3 (Flexbox and Grid layout)
* **Logic:** Vanilla JavaScript (DOM manipulation and mathematical evaluation)
